説明

タスク管理クラス

タスク継承クラスのリストを管理し、描画、更新、ウィンドウメッセージ等の配信を行う。

実行中に例外が起こったとき、どのクラスが例外を起こしたのかをログに吐き出す。 その際に実行時型情報からクラス名を取得しているので、コンパイルの際には 実行時型情報(RTTIと表記される場合もある)をONにすること。
